# PnL.fun

**Live:** [pnldotfun.vercel.app](https://pnldotfun.vercel.app)

A trading terminal style PnL visualizer.

## Stack

- Next.js 16
- Tailwind CSS 4
- Framer Motion

## Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000)
